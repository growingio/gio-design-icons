# 快速开始

## 安装

使用图标组件，你需要安装 `@gio-design/icons` 图标组件包：

<TabsState initial="yarn">
  <div id="yarn" title="Yarn">

```bash
$ yarn add @gio-design/icons
```

  </div>
  <div id="npm" title="NPM">

```bash
$ npm install --save @gio-design/icons
```

  </div>
</TabsState>

## 使用

```javascript
import React, { ReactDOM } from 'react';
import { AddTo } from '@gio-design/icons';

ReactDOM.render(
  <>
    <AddTo />
  </>,
  mountNode
);
```
